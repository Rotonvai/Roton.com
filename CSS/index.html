Introduction to CSS

CSS (Cascading Style Sheets) is used to style and layout web pages â€” for example, to alter the font, color, size, and spacing of your content.

What is CSS?

CSS is a stylesheet language that describes how HTML elements are displayed on screen. It allows you to:

  • Control colors, fonts, and layouts
  • Create responsive designs for all devices
  • Add animations and transitions
  • Build modern, interactive user interfaces
Example: CSS Basics 
/* This is a CSS comment */
p {
  color: blue;
  font-size: 16px;
  margin: 10px 0;
}

.highlight {
  background-color: yellow;
  padding: 5px;
}

CSS Syntax & Selectors

Understanding CSS syntax and selectors is fundamental to writing effective stylesheets.

CSS Rule Structure

A CSS rule consists of a selector and a declaration block:

  • Selector: targets HTML elements
  • Property: the style attribute you want to change
  • Value: the new value for the property
CSS Selectors 
/* Element selector */
p { color: black; }

/* Class selector */
.intro { font-size: 18px; }

/* ID selector */
#header { background-color: navy; }

/* Descendant selector */
div p { margin: 5px; }

/* Attribute selector */
input[type="text"] { padding: 8px; }

/* Multiple selectors */
h1, h2, h3 { font-family: Arial; }

Colors & Backgrounds

Learn how to use colors and background images to style your web pages.

Color Methods

CSS supports multiple color formats:

  • Named colors: red, blue, green
  • Hex values: #FF5733, #6366f1
  • RGB: rgb(255, 87, 51)
  • HSL: hsl(12, 100%, 60%)
  • RGBA/HSLA with transparency
Colors & Backgrounds 
/* Text color */
.text-blue { color: #3b82f6; }

/* Background color */
.bg-light { background-color: #f3f4f6; }

/* Gradient background */
.gradient {
  background: linear-gradient(
    135deg,
    #667eea 0%,
    #764ba2 100%
  );
  color: white;
  padding: 20px;
}

/* Background image */
.bg-image {
  background-image: url('image.jpg');
  background-size: cover;
  background-position: center;
}

Box Model

The CSS box model is the foundation of layout. Every element is a box composed of content, padding, border, and margin.

Box Model Components

  • Content: The actual content (text, images)
  • Padding: Space inside the border
  • Border: The edge around the padding
  • Margin: Space outside the border
Box Model Example 
.box {
  width: 300px;
  height: 150px;
  
  /* Content */
  background-color: #e0e7ff;
  
  /* Padding - inside the border */
  padding: 20px;
  
  /* Border */
  border: 3px solid #6366f1;
  
  /* Margin - outside the border */
  margin: 20px 0;
  
  /* Box sizing */
  box-sizing: border-box;
}

Text Styling

Text Properties

Fonts & Typography

Choose and style fonts effectively to improve readability and design.

Font Families

Web-safe fonts, system fonts, and web fonts via @import or @font-face

  • Serif fonts: Georgia, Times New Roman
  • Sans-serif fonts: Arial, Helvetica, Verdana
  • Monospace fonts: Courier, Courier New
  • Google Fonts and custom web fonts
Font Styling 
/* Font family stack */
body {
  font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
  font-size: 16px;
  line-height: 1.6;
}

/* Bold and italic */
.bold { font-weight: 700; }
.italic { font-style: italic; }
.light { font-weight: 300; }

/* Font variants */
.small-caps {
  font-variant: small-caps;
}

/* Web fonts (Google Fonts) */
@import url('https://fonts.googleapis.com/css2?family=Playfair+Display&display=swap');
.serif-heading {
  font-family: 'Playfair Display', serif;
  font-size: 32px;
}

Margin, Padding & Borders

Master spacing and borders to create well-structured layouts.

Spacing Basics

  • Margin: Creates space outside an element
  • Padding: Creates space inside an element
  • Border: Creates a boundary around an element
Margin, Padding & Borders 
/* Margin - shorthand */
.element {
  margin: 10px;           /* All sides */
  margin: 10px 20px;      /* Top/Bottom, Left/Right */
  margin: 10px 20px 30px 40px; /* Top, Right, Bottom, Left */
}

/* Padding - individual sides */
.box {
  padding-top: 20px;
  padding-right: 15px;
  padding-bottom: 20px;
  padding-left: 15px;
}

/* Borders */
.card {
  border: 2px solid #6366f1;
  border-radius: 8px;
  padding: 20px;
  margin: 10px 0;
}

.dashed {
  border: 2px dashed #ec4899;
}

Display, Position & Z-index

Control how elements are displayed and positioned on the page.

Display Values

  • block - Full width, starts new line
  • inline - Only takes needed width
  • inline-block - Hybrid of inline and block
  • flex - Flexible box layout
  • grid - Grid layout
  • none - Hidden from layout
Display & Position 
/* Display */
.block-element { display: block; }
.inline-element { display: inline; }
.hidden { display: none; }

/* Position */
.static { position: static; } /* default */
.relative { position: relative; top: 10px; left: 10px; }
.absolute { position: absolute; top: 50px; left: 50px; }
.fixed { position: fixed; top: 0; right: 0; }
.sticky { position: sticky; top: 0; }

/* Z-index */
.layer1 { position: relative; z-index: 1; }
.layer2 { position: relative; z-index: 2; }
.layer3 { position: relative; z-index: 3; }

Flexbox

Create flexible, responsive layouts using CSS Flexbox.

Flexbox Basics

  • Container: Parent element with display: flex
  • Items: Direct children of flex container
  • Main axis: Primary axis (default: horizontal)
  • Cross axis: Perpendicular to main axis
Flexbox Example 
.flex-container {
  display: flex;
  justify-content: center;    /* Align horizontally */
  align-items: center;        /* Align vertically */
  gap: 20px;                  /* Space between items */
  flex-wrap: wrap;            /* Wrap items */
}

.flex-item {
  flex: 1;                    /* Grow equally */
  min-width: 200px;
  padding: 20px;
  background: #e0e7ff;
  border-radius: 8px;
}

/* Flex properties */
.grow { flex-grow: 2; }
.shrink { flex-shrink: 0; }
.basis { flex-basis: 300px; }

CSS Grid

Master CSS Grid for advanced two-dimensional layouts.

Grid Basics

  • Container: Parent with display: grid
  • Rows & Columns: Define with grid-template-columns/rows
  • Gap: Space between grid cells
  • Auto-placement: Automatic item positioning
CSS Grid Example 
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);  /* 3 equal columns */
  grid-template-rows: auto;                /* Auto height rows */
  gap: 20px;                               /* Space between items */
  padding: 20px;
}

.grid-item {
  background: #e0e7ff;
  padding: 20px;
  border-radius: 8px;
  text-align: center;
}

/* Spanning multiple cells */
.span-2 { grid-column: span 2; }
.span-full { grid-column: 1 / -1; }

/* Responsive grid */
@media (max-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr;
  }
}

Transitions & Animations

Add smooth motion and interactive effects to your designs.

Transitions vs Animations

  • Transitions: Smooth change from one state to another
  • Animations: Complex sequences with keyframes
  • Timing functions: ease, linear, ease-in, ease-out
  • Duration, delay, and iteration count
Transitions & Animations 
/* Transition */
.button {
  background: #6366f1;
  transition: all 0.3s ease;
}

.button:hover {
  background: #4f46e5;
  transform: scale(1.05);
}

/* Keyframe animation */
@keyframes slideIn {
  from {
    opacity: 0;
    transform: translateX(-50px);
  }
  to {
    opacity: 1;
    transform: translateX(0);
  }
}

.animated {
  animation: slideIn 0.6s ease forwards;
}

Media Queries & Responsive Design

Create designs that work on all devices with media queries.

Responsive Breakpoints

  • Mobile: 320px - 480px
  • Tablet: 481px - 768px
  • Desktop: 769px - 1024px
  • Large Desktop: 1025px+
Media Queries 
/* Base styles (mobile-first) */
.container { width: 100%; padding: 10px; }
.column { width: 100%; }

/* Tablet */
@media (min-width: 768px) {
  .container { width: 750px; margin: 0 auto; }
  .column { width: 48%; float: left; }
}

/* Desktop */
@media (min-width: 1024px) {
  .container { width: 960px; }
  .column { width: 32%; }
}

/* Print styles */
@media print {
  .no-print { display: none; }
  body { font-size: 12pt; }
}

Pseudo-classes & Pseudo-elements

Style elements based on their state or position using pseudo-selectors.

Common Pseudo-classes

  • :hover - When user hovers over element
  • :focus - When element has focus
  • :active - When element is being activated
  • :first-child, :last-child - Position based
  • :nth-child() - Select specific children
  • :not() - Negation pseudo-class

Common Pseudo-elements

  • ::before - Insert content before element
  • ::after - Insert content after element
  • ::first-line - Style first line of text
  • ::first-letter - Style first letter
  • ::selection - Highlight selection color
Pseudo-classes & Elements 
/* Pseudo-classes */
a:hover { color: #6366f1; }
button:active { transform: scale(0.98); }
input:focus { border-color: #6366f1; outline: none; }

/* Nth-child selector */
li:nth-child(odd) { background: #f3f4f6; }
li:nth-child(3n) { color: #6366f1; }

/* Pseudo-elements */
.card::before {
  content: '';
  display: block;
  width: 4px;
  height: 100%;
  background: #6366f1;
  position: absolute;
  left: 0;
}

p::first-letter {
  font-size: 2em;
  font-weight: bold;
  color: #6366f1;
}

::selection { background: #6366f1; color: white; }

CSS Variables (Custom Properties)

Use CSS variables to create maintainable and scalable stylesheets.

Benefits of CSS Variables

  • Reuse values across your stylesheet
  • Easy to maintain and update colors, sizes, etc.
  • Can be scoped to elements or globally
  • Can be changed with JavaScript
  • Supports fallback values
CSS Variables 
/* Define global variables */
:root {
  --primary-color: #6366f1;
  --secondary-color: #8b5cf6;
  --background: #f8fafc;
  --text-color: #1e293b;
  --border-radius: 8px;
  --spacing-unit: 8px;
}

/* Dark mode variables */
@media (prefers-color-scheme: dark) {
  :root {
    --background: #0f172a;
    --text-color: #f1f5f9;
  }
}

/* Using variables */
.button {
  background: var(--primary-color);
  color: white;
  padding: calc(var(--spacing-unit) * 2);
  border-radius: var(--border-radius);
  border: none;
}

.card {
  background: var(--background, white);
  color: var(--text-color, black);
  padding: calc(var(--spacing-unit) * 3);
  border-radius: var(--border-radius);
}
