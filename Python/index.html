Welcome to Python Learning

Your beginner-friendly guide to mastering Python programming

What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. It's one of the most popular languages for beginners and professionals alike.

  • Easy to learn and understand
  • Versatile for web development, data science, and automation
  • Large community and extensive libraries
  • Open-source and free
  • Cross-platform compatibility

Why Learn Python?

Python is used by millions of developers worldwide and is the language of choice for:

  • Web Development (Django, Flask)
  • Data Science and Machine Learning
  • Artificial Intelligence
  • Automation and Scripting
  • Game Development
  • Scientific Computing

Installation and Setup

Get Python installed on your computer

Installing Python

Follow these steps to install Python on your system:

  1. Visit www.python.org
  2. Click on the download button for the latest version
  3. Run the installer
  4. Check "Add Python to PATH" before installing
  5. Complete the installation

Verify Installation

Open your terminal or command prompt and type:

python --version

You should see the Python version number displayed.

What are Variables?

Variables are containers for storing data values. In Python, you don't need to declare the type of a variable; it's automatically inferred.

Error Handling

Managing exceptions gracefully

try-except Block

try: result = 10 / 2 print(result) except ZeroDivisionError: print("Cannot divide by zero!") except Exception as e: print(f"An error occurred: {e}")

try-except-finally

try: file = open("missing.txt") except FileNotFoundError: print("File not found!") finally: print("This always executes")

Classes and Objects

Python with Database

Connecting to and managing databases

SQLite Connection

import sqlite3 # Connect to database conn = sqlite3.connect('database.db') cursor = conn.cursor() # Create table cursor.execute('''CREATE TABLE users (id INTEGER PRIMARY KEY, name TEXT, age INTEGER)''') # Insert data cursor.execute("INSERT INTO users VALUES (1, 'Alice', 30)") conn.commit() # Query data cursor.execute("SELECT * FROM users") rows = cursor.fetchall() conn.close()

Database Operations

  • CREATE - Create tables
  • INSERT - Add data
  • SELECT - Retrieve data
  • UPDATE - Modify data
  • DELETE - Remove data
  • DROP - Delete tables

Project 1: Calculator

def calculator(a, b, operation): if operation == '+': return a + b elif operation == '-': return a - b elif operation == '*': return a * b elif operation == '/': return a / b print(calculator(10, 5, '+')) # 15 print(calculator(10, 5, '*')) # 50

Project 2: Number Guessing Game

import random secret = random.randint(1, 100) guess = 0 while guess != secret: guess = int(input("Guess a number (1-100): ")) if guess < secret: print("Too low!") elif guess > secret: print("Too high!") else: print("Correct!")
