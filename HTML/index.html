Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It provides the structure and content of websites.

What is HTML?

HTML uses a system of tags and elements to tell web browsers how to display content. Every element serves a specific purpose, from defining headings to creating links and embedding images.

Why Learn HTML?

  • Foundation for web development
  • Works on all devices and browsers
  • Easy to learn for beginners
  • Essential for web designers and developers

Basic HTML Structure

Every HTML document follows a standard structure. Understanding this structure is fundamental to creating web pages.

The Essential Elements

<!DOCTYPE html>
Declares that this is an HTML5 document
<html>
The root element that wraps all content
<head>
Contains metadata, title, and links to stylesheets
<body>
Contains all visible page content

Images

The <img> tag is used to embed images in web pages. It's a self-closing element requiring specific attributes.

Image Attributes

src
Specifies the path to the image file (required)
alt
Alternative text describing the image (required for accessibility)
width & height
Set the image dimensions in pixels

Semantic HTML Tags

Semantic tags clearly define the meaning of content, improving accessibility and SEO. They tell both browsers and developers what content they contain.

Common Semantic Tags

<header>
Defines the header section of a page or article
<nav>
Contains navigation links
<article>
Represents independent content
<section>
Groups related content together
<aside>
Contains sidebar or supplementary content
<footer>
Defines the footer section

HTML5 New Elements

HTML5 introduced many new semantic and functional elements that improve code structure and user experience.

New HTML5 Elements

  • <canvas> - For drawing graphics
  • <time> - Represents a specific date/time
  • <mark> - Highlights text
  • <progress> - Shows task progress
  • <meter> - Represents a measurement
  • <figure> and <figcaption> - For illustrations and captions
